This plot reveals a new generator called "Field". The primitive presented here is the nested triangles.

The physical is a grid of 4x2 frames that can be used to produce a 8 frames loop video.

As always, it's by making a lot of prototypes that we can verify plotting constraints. I take this opportunity to share a series of reveals that will show main features of "Plottable Field (8 frames)" – my next plottable generative art on fx(hash) platform.

As a generative plotter artist, I use code to generate art (creative coding) and physically create it with pen plotters, which is itself a generative process – each physical plot is a unique variant.
